巨大惑星表層には
赤道域の幅の広いジェットと
中緯度域の幅の狭いジェットによる帯状構造，
一方，極域では渦による構造が存在することが知られている．
%%%
%
帯状構造や極渦のような  %1 逆カスケードによって，add
大規模構造は大気中の小規模乱流から形成される可能性がある(\citep{Vallis2017}．
%
このような小規模乱流を
引き起こす候補として，
雷雲が考えられている(\cite{Gierasch2000}, \cite{Ingersoll2000})． 

% 
\cite{Showman2007}, \cite{Brueshaber2019} は
この雷雲を想定した質量強制を加えた浅水実験を行った．
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% parallel
% 
% ジェットは... ジェットを主語に 
%                   % 緯度 0 - 70 度, 経度 0 - 120 度
\cite{Showman2007} は緯度$0^\circ - 70^\circ$, 
経度$0^\circ - 120^\circ$ の領域を計算した．
その結果，帯状構造が形成し，これまでの
減衰性乱流実験(\cite{Cho1996})と同様，
赤道域では東風ジェットが形成する結果が得られた\citep{Showman2007}．
%
% 渦は.... 渦を主語に
%                           緯度 60 - 90 度
\cite{Brueshaber2019} は緯度$60^\circ$ から高緯度の極域を計算した．
その結果，極渦が形成し，極渦のレジームは
Burger 数(変形半径と惑星半径の比で書かれる無次元量) によって，
最も影響を受けることがわかった．
%   具体的な渦の状態 
Burger 数が小さい場合，木星でみられる比較的小さな複数の渦が形成し，
Burger 数が大きくなるにつれて，土星などで
見られるような単一の低気圧性渦が形成することがわかった．
彼らはBurger 数の値によって，変化する渦のレジームを
観測されている巨大惑星の極渦の特徴と比較し，
木星的，遷移，土星的，巨大氷惑星的レジームと分類した．

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% parallel
%
% 段落をかえて，本研究の目的は... と1つの段落に

現実の雷雲は全球的に発生しているが，
どちらの先行研究も領域計算である．
%
そのため，全球計算を行い，
彼らの結果を確かめる必要がある．
%
なぜなら，先行研究では考慮されていない
領域外の運動量輸送などにより，
計算結果で得られたジェットや
渦の構造が変化する可能性があるからである．
%
% 渦の強さではダメ?  上で述べた構造について，言えたらいい
%(例えば，\cite{Brueshaber2019} では
%計算領域の境界付近で速度場を減衰させている．
%中緯度からの運動量輸送が考慮されていなく，
%渦の強化を妨げる可能性がある．)．
本研究の目的は雷雲を想定して与えた質量強制による
数値実験を全球で行い，
雷雲による強制により，巨大惑星で見られる
帯状構造と極渦が形成されるのかを調べることである．                               
また，それらの構造を先行研究で得られた結果と比較し，
領域計算で得られた結果の妥当性を調べる．
% どういう東西風分布か？詳細に
% 極渦が出来るから，レジームが変化するかどうか．
%
%

本研究はスペクトル法(地球流体電脳倶楽部の階層的地球スペクトルモデル集
(SPMODEL; \cite{spmodel2006}, \cite{spmodel2013}))を用い，
雷雲を想定した質量強制を加えた1.5層浅水系全球計算を行う．
% ここに，使ったモデルの説明. SPMODEL を使って，1.5層浅水系 
% 
% Burger 数が大きいと，渦がこう，ジェットがこう．小さいと渦がこう，ジェットがこう.
% これはロバストな構造でalpha , R_Storm を変更してもかわらない.
%実験は次の4つのパラメータ % 実験を調べる.
%Burger 数(実験1)，局所的な放射緩和の強さ(実験2)，
%質量強制の正負の割合 (実験3)，質量強制の空間的大きさ (実験4) の影響を調べる． % 影響=> 依存性
%
% 渦はこう，ジェットはこう． parallel
%%%
計算の結果，全球計算でも
\cite{Showman2007} で得られた帯状構造と
\cite{Brueshaber2019} で得られた渦の構造が形成するという結果が得られた．
% jet
また，帯状構造や極渦は質量強制の正負の割合や，
空間的大きさを変更しても変わらないことがわかった．
%
そして，極渦のレジームは今回変更した他のパラメータに比べ，
Burger 数の値に最も影響を受けることがわかった．
%
これらの結果は，\cite{Showman2007}, \cite{Brueshaber2019} で
得られた結果と整合的である．
%
しかし，今回の計算結果では赤道域での風速が\cite{Showman2007} の結果に比べて，
約10倍大きくなることがわかった．
これは，\cite{Showman2007} では計算できていない領域からの影響を示唆している．
