%%%
巨大惑星表層の風速分布は
赤道域の幅の広いジェットや
帯状構造という特徴がある．
%
また，極域では低気圧性渦が支配的で
そのレジームが異なることが知られている．
%

このような風速分布と極渦を形成する
エネルギー源の1つとして，雷雲が考えられている．
\cite{Showman2007}, \cite{Brueshaber2019} は
雷雲を想定した質量強制を加えた浅水実験を行った．
%
\cite{Showman2007} は低緯度から中緯度の領域を計算し，
中緯度の帯状構造と赤道域の東風ジェットが形成されることを確認した．
%
一方，\cite{Brueshaber2019} は極域の領域を計算し，
Burger 数と呼ばれる変形半径と
惑星半径の比で表される無次元量の値によって，
極渦のレジームが異なることがわかった．

どちらの先行研究も領域計算であり，
このような領域の制限は流れ場全体の構造に
影響を与える可能性がある(例えば，\cite{Showman2007}では
南半球からの運動量輸送が考慮できていない．)
%(例えば，\cite{Brueshaber2019} では
%計算領域の境界付近で速度場を減衰させている．
%中緯度からの運動量輸送が考慮されていなく，
%渦の強化を妨げる可能性がある．)．

本研究の目的は彼らが与えた質量強制の球面浅水実験の全球計算を行い，
巨大惑星の風速分布や極渦が雷雲による強制により，形成されるのかを調べることである．
また，低中緯度と極域の構造を先行研究と比較する．

実験は次の4つのパラメータ
Burger 数(実験1)，局所的な放射緩和の強さ$\tau_{\rm{APE}}$ (実験2)，
質量強制の正負の割合$\alpha$(実験3)，質量強制の空間的大きさ$R_{\rm{storm}}$(実験4) 
の影響を調べる．

計算の結果，他のパラメータに比べ，
Burger 数の値によって，極渦のレジームが変化することがわかった．
これは\cite{Brueshaber2019} の結果と整合的である．

東西風の分布はジェットが発達する全てのケースで
赤道域の東風ジェットと中緯度域の西風ジェットが形成する．
%
\cite{Showman2007} と同様のパラメータを用いて，計算した結果，
\cite{Showman2007} の計算結果に比べて，
赤道での風速の大きさが約10倍大きくなることがわかった．
これは，\cite{Showman2007} では考慮されていない，
南半球からの運動量輸送の影響を示唆している．
