巨大惑星表層には
赤道域の幅の広いジェットや
中緯度域での帯状構造，
極域での極渦が特徴として知られている．
%%%
%
このような大規模構造は
逆カスケード効果によって，大気中の小規模乱流から
形成されることが知られている\citep{Vallis2017}．
%
このような小規模乱流を
引き起こす候補として，
雷雲が考えられる(\cite{Gierasch2000}, \cite{Ingersoll2000})． 

\cite{Showman2007}, \cite{Brueshaber2019} は
この雷雲を想定した質量強制を加えた浅水実験を行った．
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% parallel
% 
% ジェットは... ジェットを主語に 
%                   % 緯度 0 - 70 度, 経度 0 - 120 度
緯度$0^\circ - 70^\circ$, 経度$0^\circ - 120^\circ$ の領域を計算した結果，
赤道域の幅の広いジェットや帯状構造が形成した．
しかし，赤道でのジェットは減衰性乱流実験\citep{Cho1996} の結果と同様，
土星，木星で見られるような赤道西風ジェットは出来なかった\citep{Showman2007}． 
% 減衰性乱流などの先行研究と同様に東風ジェットになることを確認した?  
%
% 渦は.... 渦を主語に
%                           緯度 60 - 90 度
一方，緯度$60^\circ$ から高緯度の極域を計算し結果，極渦が形成した．
また，その極渦の大きさや数といったレジームは
Burger 数(変形半径と惑星半径の比で書かれる無次元量) によって，影響を受けることがわかった．
%   具体的な渦の状態 
Burger 数が小さい場合，木星でみられる比較的小さな複数の渦が形成し，
Burger 数が大きくなるにつれて，土星などで
見られるような単一の低気圧性渦が形成することがわかった．
彼らはBurger 数の値によって，変化する渦のレジームを
観測されている巨大惑星の極渦の特徴を比較し，
木星的，遷移，土星的，巨大氷惑星的レジームと分類した．
%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% parallel
%
% 段落をかえて，本研究の目的は... と1つの段落に

どちらの先行研究も領域計算であり，
\cite{Showman2007} は南半球を\cite{Brueshaber2019} は
緯度 $60^\circ$ から低緯度からの影響を全く考慮出来てない．
雷雲は全球的に発生しているため，
雷雲による影響を調べるためには全球計算が必要になる．
% 渦の強さではダメ?  上で述べた構造について，言えたらいい
%(例えば，\cite{Brueshaber2019} では
%計算領域の境界付近で速度場を減衰させている．
%中緯度からの運動量輸送が考慮されていなく，
%渦の強化を妨げる可能性がある．)．
本研究の目的は彼らが与えた質量強制の球面浅水実験の全球計算を行い，
雷雲による強制により，巨大惑星で見られる
赤道域の幅の広いジェットと
中緯度の帯状構造，
極渦が形成されるのかを調べることである．                               
% どういう東西風分布か？詳細に
% 極渦が出来るから，レジームが変化するかどうか．
また，東西風のジェットの構造と
極渦のレジームを先行研究と比較する．
%
%
我々はスペクトル法(地球流体電脳倶楽部の階層的地球スペクトルモデル集
(SPMODEL; \cite{spmodel2006}, \cite{spmodel2013}))を用い，
雷雲を想定した質量強制を加えた1.5層浅水系全球計算を行った．
% ここに，使ったモデルの説明. SPMODEL を使って，1.5層浅水系 
% 
% Burger 数が大きいと，渦がこう，ジェットがこう．小さいと渦がこう，ジェットがこう.
% これはロバストな構造でalpha , R_Storm を変更してもかわらない.

%実験は次の4つのパラメータ % 実験を調べる.
%Burger 数(実験1)，局所的な放射緩和の強さ(実験2)，
%質量強制の正負の割合 (実験3)，質量強制の空間的大きさ (実験4) の影響を調べる． % 影響=> 依存性
%
% 渦はこう，ジェットはこう． parallel
%%%
計算の結果，
% jet
赤道域の東風ジェットと中緯度域の西風ジェットが形成することがわかった．
これはロバストな構造で，質量強制の正負の割合や，
空間的大きさを変更しても変わらないことがわかった．
このジェットの構造は
\cite{Showman2007} で得られた結果と整合的であるが，
我々の計算結果では赤道域での風速が約10倍大きくなることがわかった．
これは，\cite{Showman2007} では考慮されていない，
南半球からの影響を示唆している．
% vor
極域では渦が形成することがわかった．
極渦のレジームは今回変更し
た他のパラメータに比べ，Burger 数の値に最も影響をうけ，
\cite{Brueshaber2019} で分類されたものと整合的ある．
