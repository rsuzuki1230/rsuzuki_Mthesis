エネルギー源の1つとして，雷雲が考えられている． % エネルギー源とは? 要確認. 人のせいにする．

\cite{Showman2007} は低緯度から中緯度の領域を計算し，  
中緯度の帯状構造と赤道域の東風ジェットが形成されることを確認した．  % 減衰性乱流などの先行研究
%                                                                    %と同様に東風ジェットになることを確認した?  
%

%                           緯度 60 - 90 度
一方，\cite{Brueshaber2019} は極域の領域を計算し，　
Burger 数と呼ばれる変形半径と
惑星半径の比で表される無次元量の値によって，
%  
%   具体的な渦の状態 
%
極渦のレジームが異なることがわかった．
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%% parallel
%
% 段落をかえて，本研究の目的は... と1つの段落に
どちらの先行研究も領域計算であり，
このような領域の制限は流れ場全体の構造に
影響を与える可能性がある(例えば，\cite{Showman2007}では % ()はダメ.
南半球からの運動量輸送が考慮できていない)． % 渦の強さではダメ?  上で述べた構造について，言えたらいい
%(例えば，\cite{Brueshaber2019} では
%計算領域の境界付近で速度場を減衰させている．
%中緯度からの運動量輸送が考慮されていなく，
%渦の強化を妨げる可能性がある．)．

本研究の目的は彼らが与えた質量強制の球面浅水実験の全球計算を行い，
巨大惑星の極渦や東西風分布が雷雲による強制により， % どういう東西風分布か？詳細に
形成されるのかを調べることである．                 % 極渦が出来るから，レジームが変化するかどうか．
また，低中緯度と極域の構造を先行研究と比較する．
%
% ここに，使ったモデルの説明. SPMODEL を使って，1.5層浅水系 , 解像度
% 
% Burger 数が大きいと，渦がこう，ジェットがこう．小さいと渦がこう，ジェットがこう.
% これはロバストな構造でalpha , R_Storm を変更してもかわらない.
実験は次の4つのパラメータ % 実験を調べる.
Burger 数(実験1)，局所的な放射緩和の強さ(実験2)，
質量強制の正負の割合 (実験3)，質量強制の空間的大きさ (実験4) の影響を調べる． % 影響=> 依存性

% 渦はこう，ジェットはこう． parallel
計算の結果，極域で低気圧性渦が形成し，
そのレジームは他のパラメータに比べ，
Burger 数の変化に影響をうけることがわかった．
これは\cite{Brueshaber2019} の結果と整合的である．
%
また，東西風の分布はジェットが発達する全てのケースで
赤道域の東風ジェットと中緯度域の西風ジェットが形成する．
%
\cite{Showman2007} と同様のパラメータを用いて，計算した結果，
\cite{Showman2007} の計算結果に比べて，
赤道域での風速の大きさが約10倍大きくなることがわかった．
これは，\cite{Showman2007} では考慮されていない，
南半球からの運動量輸送の影響を示唆している．
